# Two Leaves and a Bud - Website Clone 🍵

A fully responsive frontend clone of the e-commerce website [Two Leaves and a Bud](https://www.twoleavestea.com/). This project was built entirely using standard web technologies to demonstrate UI/UX implementation without relying on JavaScript.

## 🚀 Tech Stack
* **HTML5:** Semantic structuring and clean markup.
* **CSS3:** Custom styling, Flexbox, and CSS Grid for complex layouts.

## ✨ Key Features
* **100% Responsive Design:** Adapts perfectly to desktop, tablet, and mobile screens.
* **Pure CSS Carousels:** Implemented swipable review and product carousels using `scroll-snap` and `overflow-x` (Zero JavaScript used).
* **Hover Animations:** Smooth transitions and interactive hover effects on buttons, product cards, and navigation links.
* **Modern Layouts:** Recreated the intricate Hero, Best Sellers, and Discover sections with pixel-perfect accuracy.
